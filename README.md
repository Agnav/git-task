# Team Profile Git Task

This task is designed to help you practice a basic Git workflow used in real development teams.

You will learn how to:

- Clone a repository
- Create a new branch
- Make changes to a project
- Commit your changes
- Push your branch to GitHub
- Create a Pull Request (PR)

---

# Task Objective

Add your own profile card to the `index.html` file.

Example:

```html
<div>
  <h2>Your Name</h2>
  <p>Your Role</p>
</div>
```

---

# Step 1 — Clone the Repository


# Step 2 — Create a New Branch

Create a feature branch for your changes.

Branch name format:

```bash
feature/your-name
```

---

# Step 3 — Make Your Changes

Open the project in your editor.

Edit:

```text
index.html
```

Add your own profile section below the existing profile card.

Example:

```html
<div>
  <h2>Jane Smith</h2>
  <p>Backend Developer</p>
</div>
```

Save the file.

---

# Step 4 — Check Git Status

Run:

```bash
git status
```

This shows which files were modified.

---

# Step 5 — Stage Your Changes

  Check status again:

  You should now see the files marked in green.

---

# Step 6 — Commit Your Changes

Create a commit with a meaningful message.

---

# Step 7 — Push Your Branch

Push your branch to GitHub.

---

# Step 8 — Create a Pull Request (PR)

Go to the GitHub repository in your browser.

GitHub should show a button:

```text
Compare & pull request
```

Click it.

Then:

1. Add a PR title
2. Add a short description
3. Click "Create Pull Request"

---

# Task Completion Checklist

- [ ] Repository cloned
- [ ] New branch created
- [ ] Profile card added
- [ ] Changes committed
- [ ] Branch pushed
- [ ] Pull Request created

---

# Important Notes

- Do not commit directly to the `main` branch.
- Always work inside your own feature branch.
- Write clear commit messages.
- Ask questions if you get stuck.
